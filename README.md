# polynomial-root-webservice
Webservice to find a root of a polynomial using Newton-Raphson method

### e.g:  $6-x-7x^2+x^3+x^4$

**Request:**
<br>
GET /solve?coefficients=6+-1+-7+1+1&guess=2.5
<br>
**Response:**
<br>
{"root":2.0}
<br>
